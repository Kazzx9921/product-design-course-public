# Landing Page Starter

這是「從想法到收款」課程 Chapter 1 的實作範本。

## 開始使用

```bash
git clone https://github.com/Kazzx9921/product-design-course-public.git my-landing-page
cd my-landing-page
npm install
npm run dev
```

## 把你的 AI HTML 轉成 Next.js

1. 開啟 Claude Code：`claude`
2. 執行指令：`/html-to-page`
3. 貼入你的 HTML 內容
4. 完成後執行 `npm run dev` 預覽結果

## 部署到 Zeabur

1. Push 到你的 GitHub repo
2. 前往 [zeabur.com](https://zeabur.com) 連接 repo
3. 選擇 Next.js 框架，一鍵部署
