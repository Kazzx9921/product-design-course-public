# Landing Page Starter

這是「從想法到收款」課程 Chapter 1 的實作範本。

## 開始使用

1. 下載並解壓縮這個 repo
2. 進入資料夾，安裝套件：
   ```bash
   cd product-design-course-public-main
   npm install
   ```
3. 開啟 Claude Code：`claude`

## 把你的 AI HTML 轉成 Next.js

1. 將你的 HTML 存入 `docs/Landing.html`
2. 在 Claude Code 中執行：`/html-to-page`
3. 執行 `npm run dev` 預覽結果（http://localhost:3000）

## 推上你自己的 GitHub

在 Claude Code 中執行 `/setup`，Claude 會引導你：
- 初始化 git
- 建立你自己的 GitHub repo
- Push 上去

## 部署到 Zeabur

1. 前往 [zeabur.com](https://zeabur.com) 連接你的 GitHub repo
2. 選擇 Next.js 框架，一鍵部署
